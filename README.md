# TodoSpringboot
간단한 todo 프로젝트를 Spring boot와 Mysql 을 이용해서 CRUD API 구현
<br/><br/>

<p><b>[사용된 기능]</b></p>

* DB : Mysql (JPA)
* 백엔드 : Spring boot 
* 문서 : Swagger
